# Hypr-Daze
Hyprland Dotfiles 

This rice attemps to combine the best of all features, eyecandy and looks from modern pre-existing dotfiles like caelestia, end4, omarchy and HyDE. This rice is heavily inspired by them so check them out! 

These dotfiles are still work in progress. The rice is nowhere NEAR finished and I will be maintaining this repo as I progress. 
Feel free to open up any issues or give me any suggestions :)
